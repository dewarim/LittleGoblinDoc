Package de.dewarim.goblin.reputation

 

Class Summary
Faction A faction is an entity (or group) with which a player character may earn a certain Reputation.
Reputation Mapping class between PlayerCharacter and Faction.
ReputationMessage A ReputationMessage describes a player character's current reputation status with a faction in a short text, for example: "The dwarves of Deep Mountain would buy you a beer anytime.".
ReputationMessageMap The reputation with a given faction is internally an integer value.
